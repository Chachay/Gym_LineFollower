# Gym Line Follower
Simple 2D Line Follower Environment with OpenAI gym I/F.

<img src="https://raw.githubusercontent.com/Chachay/Gym_LineFollower/master/Demo.png">


## Requirement
- Python 2.7
- OpenAI-Gym (https://github.com/openai/gym)
- numpy
- wxPython3.0-win32-py27

## Usage
```
python example.py
```
